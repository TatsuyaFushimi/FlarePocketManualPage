STEP 1

初回セットアップ（初回のみ）

① GitHubにログインする

ターミナルで以下を実行してください：

gh auth login
💡 ブラウザが開いてコードの入力を求められます。ターミナルに表示されたコード（XXXX-XXXX形式）をブラウザに貼り付けてください。会社から招待されたGitHubアカウントでログインしてください。

② リポジトリをクローンする

ターミナルで以下を実行してください：

git clone https://github.com/EarthShipJP/FlarePocketPJ.git

③ Cursorで開いてセットアップする

クローンしたフォルダをCursorで開き、ClaudeCodeに以下をコピペしてください：

このプロジェクトのセットアップをしてください。
- ffmpegが入っていなければインストール
- .envファイルを作成してREVIEWER_NAME=（自分の名前ローマ字）を設定
- pip3 install -r requirements_web.txt を実行
💡 途中で「許可しますか？」などのYES/NO確認が出た場合は、すべて YES で進めてください。
STEP 2

添削を依頼する

YouTube URL の場合

（YouTubeのURL）を添削してください

動画ファイルの場合

MP4ファイルを input/ フォルダに置いてからコピペ：

input/（ファイル名）.mp4 を添削してください
⏱ 処理時間の目安：30分の動画で約15〜20分。完了するとレポートが output/backlog/ に保存されます。
💡 処理中にYES/NOの確認が出た場合は、すべて YES で進めてください。
STEP 3

レポートを確認・フィードバックする

添削完了後、ClaudeCodeがレポートを表示します。誤検出や見逃しがあればそのまま伝えてください：

誤検出の場合

○○:○○の指摘は誤検出です

見逃しがある場合

○○:○○に見逃しがあります：（内容）
📝 フィードバックはAIが自動で FlarePocket_brain.md に反映し、GitHubにpushします。次回以降の添削から全員が最新ルールで動きます。

チェックルール一覧

コード内容
HY1誤字・固有名詞の誤表記
HY2口語表現・語彙ミス・助詞抜け
HY4全角数字→半角
HY6改行位置・短すぎるテロップ
MOモザイク漏れ（ASIN・個人情報）
SE演出テロップなし
CAバナー・ロゴ未表示
TX2カット候補（相槌のみ等）

詳細ルールは FlarePocket_brain.md を参照。

STEP 4

添削を始める前に（毎回）

最新のルールを取得するため、添削を依頼する前に必ず以下をコピペしてください：

添削の前にgit pullで最新のbrainを取得してください
📥 他のユーザーが更新したルールが自動で反映されます。
