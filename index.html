FlarePocket 使用方法

GitHubリポジトリを開く
STEP 0

プロジェクトの取得（初回のみ）

① ターミナルを開く

キーボードで ⌘（command）+ スペース を押してSpotlightを開き、「ターミナル」と入力してEnterを押してください。黒っぽいウィンドウが開けばOKです。

② 保存先フォルダを指定する

プロジェクトを保存したいフォルダをターミナルに指定します。以下の手順で行ってください：

1

ターミナルに以下を入力する（コピペでOK）

※ cd の後ろに半角スペースが1つ入っています

cd
2

Finderで保存したいフォルダを開く

デスクトップやDocumentsなど、わかりやすい場所を選んでください。

3

フォルダ名の横にある 🗂️ をターミナルにドラッグ＆ドロップ

Finderのウィンドウ上部、フォルダ名の左横に小さな 🗂️ アイコンがあります。それをターミナルにドラッグすると、パスが自動で入力されます。

💡 🗂️ を押しながらドラッグでOKです
$ cd /Users/yourname/Desktop ← 自動で入力される
4

Enterを押す

これで保存先の指定が完了です。

③ プロジェクトをダウンロードする

以下をコピペしてEnterを押してください：

git clone https://github.com/EarthShipJP/FlarePocketPJ.git
💡 文字がたくさん流れて止まったら完了です。デスクトップに FlarePocketPJ フォルダが作成されています。

④ Cursor でフォルダを開く

Cursor を起動し、上部メニューから File → Open Folder を選択して、ダウンロードされた FlarePocketPJ フォルダを選んでください。

💡 Cursorがまだインストールされていない場合は cursor.sh からダウンロードしてください。
STEP 1

初回セットアップ（初回のみ）

CursorでClaudeCodeに以下をコピペしてください：

このプロジェクトのセットアップをしてください。
- GitHubにログインしていなければ gh auth login を実行（会社から招待されたアカウントでログイン）
- ffmpegが入っていなければインストール
- yt-dlpが入っていなければインストール
- .envファイルを作成してREVIEWER_NAME=（自分の名前ローマ字）を設定
- pip3 install -r requirements_web.txt を実行
💡 途中で「許可しますか？」などのYES/NO確認が出た場合は、すべて YES で進めてください。これが完了すると次回から確認なしで動きます。
STEP 2

添削を依頼する

YouTube URL の場合

（YouTubeのURL）を添削してください

動画ファイルの場合

MP4ファイルを input/ フォルダに置いてからコピペ：

input/（ファイル名）.mp4 を添削してください
⏱ 処理時間の目安：30分の動画で約15〜20分。完了するとレポートが output/backlog/ に保存されます。
💡 処理中にYES/NOの確認が出た場合は、すべて YES で進めてください。
STEP 3

レポートを確認・フィードバックする

添削完了後、ClaudeCodeがレポートを表示します。誤検出や見逃しがあればそのまま伝えてください：

誤検出の場合

○○:○○の指摘は誤検出です

見逃しがある場合

○○:○○に見逃しがあります：（内容）
📝 フィードバックはAIが自動で FlarePocket_brain.md に反映し、GitHubにpushします。次回以降の添削から全員が最新ルールで動きます。

チェックルール一覧

コード内容
HY1誤字・固有名詞の誤表記
HY2口語表現・語彙ミス・助詞抜け
HY4全角数字→半角
HY6改行位置・短すぎるテロップ

SE演出テロップなし
CAバナー・ロゴ未表示

詳細ルールは FlarePocket_brain.md を参照。

STEP 4

添削を始める前に（毎回）

最新のルールを取得するため、添削を依頼する前に必ず以下をコピペしてください：

添削の前にgit pullで最新のbrainを取得してください
📥 他のユーザーが更新したルールが自動で反映されます。
