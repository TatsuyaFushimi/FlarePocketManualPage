Frame.io 自動コメント投稿とは？

FlarePocket で検出した添削指摘を、Frame.io の動画にタイムスタンプ付きのコメントとして自動投稿する機能です。ボタン一つで全指摘を一括投稿できます。

2つの投稿モード

モード必要なものこんな人向け
⭐ URL共有モード（おすすめ） 編集者からもらった 共有URL だけ アカウント・トークン不要。URL共有で動画を受け取っている人
API Token モード Asset ID ＋ API Token 自分のアカウントで動画をアップロードしている人
✅ 編集者と添削者が別の人でも、URL共有モードなら共有URLを貼るだけで投稿できます。アカウントもトークンも不要です。
STEP 1（URL共有モード）

⭐ 共有URLだけで投稿する（おすすめ）

編集者から受け取った Frame.io の共有URL を貼るだけで投稿できます。アカウントもトークンも不要です。

編集者から共有URLをもらう

編集者が Frame.io で発行した共有リンクを受け取ります。以下のような形式の URL です。

https://app.frame.io/reviews/XXXXXXXXXX
https://app.frame.io/v/XXXXXXXXXX

FlarePocket で添削を完了させる

Web アプリ（http://localhost:8000）で動画を添削します。完了するとレポートが表示されます。

「URL共有（トークン不要）」タブで共有URLを貼る

レポート下部の「Frame.io エクスポート」セクションを開き、「URL共有（トークン不要）」タブ（デフォルト）を選択して共有URLを貼り付けます。

画面イメージ
URL共有（トークン不要）
API Token
Frame.io 共有URL（https://app.frame.io/reviews/XXXXX）
コメントを投稿

「コメントを投稿」ボタンを押す

ボタンをクリックすると、全指摘が一括で Frame.io に投稿されます。

✅ 「✓ N 件投稿成功」 と表示されれば完了です。
🚫 ボタンの二度押しに注意！重複してコメントが投稿されます。
STEP 2（API Token モード）

API Token で投稿する（自分のアカウントで動画をアップした人向け）

自分のアカウントで動画をアップロードしている場合は、Asset ID と API Token を使って投稿できます。

API Token を発行する（初回のみ）

Frame.io にログイン → 左下のアカウントアイコン → 「Account Settings」 → 左メニュー 「Developer」 → 「New Token」 で発行します。

⚠️ トークンは作成直後にしか表示されません。必ずコピーして保管してください。

発行したトークンを .env に追加しておくと次回から入力不要になります：

FRAMEIO_TOKEN=ここに発行したトークンを貼り付け

Asset ID を確認する

Frame.io で動画を開き、ブラウザの URL から Asset ID をコピーします。

https://app.frame.io/player/a1b2c3d4-e5f6-7890-abcd-ef1234567890
https://app.frame.io/projects/xxxx/assets/a1b2c3d4-e5f6-7890-abcd-ef1234567890

オレンジ色の部分 が Asset ID です。

「API Token」タブで入力して投稿する

レポート下部の Frame.io エクスポートセクションで 「API Token」タブ を選び、Asset ID と Token を入力して投稿します。

画面イメージ
URL共有（トークン不要）
API Token
Asset ID（Frame.io の動画アセット ID）
API Token（.env に設定済みなら空欄可）
コメントを投稿
STEP 4

Frame.io でコメントを確認する

Frame.io で動画を開く

コメントを投稿した動画を Frame.io で開きます。タイムライン下部にコメントが並んでいます。各コメントのタイムコードをクリックするとその位置に動画がジャンプします。

コメントの形式を確認する

各コメントはルールコード付きで以下の形式で投稿されます。

[HY1] テロップ「体に良いわけがない」→「体にいいわけがない」に修正
[HY2] テロップ「すごいよくなってきているといいますか」→「すごくよくなってきているといいますか」に修正
[CA] エンドカード「伊原 靖（イハラ ヤスシ」→閉じ括弧「）」が欠落。修正してください

よくある質問

動画をアップロードしたのは編集者で、添削するのは別の人です。どうすればいいですか？
URL共有モードを使えば解決します。編集者から共有URLをもらって「URL共有（トークン不要）」タブに貼るだけでOKです。アカウントもトークンも必要ありません。
共有URLがどんな形式かわかりません
Frame.io の共有URLは https://app.frame.io/reviews/XXXXX または https://app.frame.io/v/XXXXX の形式です。編集者に「Frame.io の共有リンクを送ってください」と伝えれば発行してもらえます。
「URLの形式を認識できませんでした」というエラーが出ます
貼り付けたURLが Frame.io の共有リンク形式（/reviews/ または /v/ を含むURL）でない可能性があります。編集者に共有リンクを再発行してもらってください。
「レビューリンクの情報を取得できませんでした」というエラーが出ます
共有リンクの有効期限が切れているか、URLが正しくコピーできていない可能性があります。編集者に共有リンクを再発行してもらってください。
「HTTP 401 / 403」エラーが出ます（API Token モード）
401 は Token が無効または期限切れ、403 は Asset ID が間違っているかアクセス権がありません。URL共有モードへの切り替えも検討してください。
コメントが重複して投稿されてしまいました
「コメントを投稿」ボタンを複数回押すと重複します。Frame.io 側で重複コメントを手動削除してください。
それでも解決しない場合は？
伏見に気軽に聞いてください🙌
