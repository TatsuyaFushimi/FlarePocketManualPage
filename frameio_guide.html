Frame.io 自動コメント投稿

FlarePocket で検出した添削指摘を、Frame.io の動画にタイムスタンプ付きのコメントとして自動投稿する機能です。
一度設定すれば、ボタン一つで全指摘を一括投稿できます。

📋 はじめる前に：必要なもの

必要なもの説明どこで入手？
Frame.io アカウント 動画がアップロードされているアカウント frame.io（無料プランあり）
Asset ID コメントを投稿したい動画の ID Frame.io の動画 URL から取得（STEP 2 で説明）
API Token FlarePocket が Frame.io を操作するための認証キー Frame.io のアカウント設定から発行（STEP 1 で説明）
💡 API Token は一度発行して .env に保存しておけば、次回以降は入力不要になります。

🔄 全体の流れ

① API Token を発行
→
② Asset ID を確認
→
③ FlarePocket で添削
→
④ エクスポートボタンを押す
→
✓ Frame.io にコメントが届く
STEP 1

API Token を発行する

Frame.io にログインする

ブラウザで app.frame.io を開き、アカウントにログインしてください。

アカウント設定を開く

画面左下のアカウントアイコン（プロフィール画像）をクリックし、メニューから 「Account Settings」 を選択します。

「Developer」タブを開く

左メニューの 「Developer」 をクリックします。

新しいトークンを発行する

「New Token」 ボタンをクリックし、わかりやすい名前（例：FlarePocket）を入力して作成します。

⚠️ トークンは作成直後にしか表示されません。必ずコピーして安全な場所に保管してください。

（推奨）.env に保存する

プロジェクトフォルダの .env をテキストエディタで開き、以下の行を追加します。次回から入力欄を空欄にできます。

FRAMEIO_TOKEN=ここに発行したトークンを貼り付け
✅ .env は Git にコミットされないため、安全にトークンを保管できます。
STEP 2

Asset ID を確認する

コメントを投稿したい動画を開く

Frame.io でコメントを入れたい動画を開きます。

URL から Asset ID を読み取る

ブラウザのアドレスバーを確認します。URL は以下のような形式になっています。

https://app.frame.io/player/a1b2c3d4-e5f6-7890-abcd-ef1234567890

または

https://app.frame.io/projects/xxxx/assets/a1b2c3d4-e5f6-7890-abcd-ef1234567890

オレンジ色の部分 が Asset ID です。ハイフン区切りの長い文字列（UUID 形式）になっています。

Asset ID をコピーする

URL 末尾または /player/ 以降の部分をコピーします。

a1b2c3d4-e5f6-7890-abcd-ef1234567890
💡 動画を右クリック →「Copy Asset ID」でコピーできる場合もあります（Frame.io のバージョンにより異なります）。
STEP 3

FlarePocket でエクスポートする

添削を完了させる

FlarePocket の Webアプリ（http://localhost:8000）で動画 URL を入力し、添削が完了するまで待ちます。

「Frame.io エクスポート」セクションを見つける

添削レポートの下部に 「Frame.io エクスポート」 セクションが表示されています。

画面イメージ
FRAME.IO エクスポート
Asset ID（Frame.io の動画アセット ID）
API Token（.env に FRAMEIO_TOKEN 設定済みなら空欄可）
コメントを投稿

Asset ID と API Token を入力する

入力欄入力内容
Asset ID STEP 2 で確認した UUID（例：a1b2c3d4-e5f6-…）
API Token STEP 1 で発行したトークン。.env に設定済みなら空欄でOK

「コメントを投稿」ボタンを押す

ボタンをクリックすると、全指摘が一括で Frame.io に投稿されます。

✅ 「✓ N 件投稿成功」 と表示されれば完了です。
💡 見逃し追加した指摘も一緒に投稿されます。
STEP 4

Frame.io でコメントを確認する

Frame.io で動画を開く

コメントを投稿した動画を Frame.io で開きます。タイムライン下部にコメントが並んでいます。タイムコードをクリックするとその秒数に動画がジャンプします。

コメントの形式

各コメントはルールコード付きで投稿されます。

[HY1] テロップ「体に良いわけがない」→「体にいいわけがない」に修正
[HY2] テロップ「すごいよくなってきているといいますか」→「すごくよくなってきているといいますか」に修正
[CA] エンドカード「伊原 靖（イハラ ヤスシ」→閉じ括弧「）」が欠落。修正してください

❓ よくある質問

Asset ID がどこにあるかわかりません
Frame.io でコメントしたい動画を開き、ブラウザのアドレスバーを確認してください。/player/ または /assets/ の後ろに続く長い文字列（ハイフン区切り）が Asset ID です。
「Frame.io トークンが必要です」というエラーが出ます
API Token の入力欄が空欄で、かつ .env にも FRAMEIO_TOKEN が設定されていない状態です。STEP 1 を参考にトークンを発行して入力欄に貼り付けてください。
「HTTP 401」エラーが出ます
API Token が無効または期限切れです。Frame.io の Developer 設定で新しいトークンを発行し直してください。
「HTTP 403」エラーが出ます
Asset ID が間違っているか、そのアセットへのアクセス権がありません。自分のアカウントでアップロードした動画の Asset ID であることを確認してください。
コメントが重複して投稿されてしまいました
「コメントを投稿」ボタンを複数回押すと重複します。Frame.io 側で重複コメントを手動削除してください。投稿済みジョブには「エクスポート済み」バッジが表示されるので、次回からは確認してから押してください。
それでも解決しない場合は？
伏見に気軽に聞いてください🙌
